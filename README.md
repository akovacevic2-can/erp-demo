# Gearbox Costing App Prototype

Static GitHub-testable prototype for gearbox costing, BOM management, work orders, routing, and dashboard overview.

## Latest version marker

`v2026-06-02-latest-dashboard-workorders-routing-bom-machines`

Search for this text in `app.js` to confirm you uploaded the newest revision.

## Run locally

Open `index.html`, or run:

```bash
python -m http.server 8000
```

Then open <http://localhost:8000>.

## GitHub Pages

Upload these files to the repository root, not inside another folder:

- `index.html`
- `app.js`
- `styles.css`
- `package.json`
- `README.md`
- `.github/workflows/pages.yml`

The app stores demo edits in browser local storage.
